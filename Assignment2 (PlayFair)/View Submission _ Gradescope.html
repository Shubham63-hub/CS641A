Q4 Analysis

20 Points

What tools and observations were used to figure out the cryptosystem? (Explain in less than 300 words)

When we started the level, we got a text that talked about some patterns written on the boulder, so we tried the read command and got the ciphertext. We first assumed it to be a substitution cipher, and we employed frequency analysis on it, but even after multiple tries, we did not get any clue about the text. Then, we tried Caesar cipher, but that also did not work. Then we thought we might get hints from the text, which appeared at the starting off level. In the last, it was mentioned that "some patterns are written on one of the distant boulders . The word 'distant' gave us a hint that we should try the command "go." After the go command, some text appeared on the screen, which contained dots and dashes. Due to prior knowledge, we confirmed it is the MORSE code, then we looked on the internet and translated it to the word "CRYPTANALYSIS" because Morse code is simply a substitution cipher. The last line of text was written in the capital, "PLAY FAIR." After googling the word "play fair", we learned that "play fair" itself is an encryption technique. After watching the tutorial on the "Playfair" encryption method that requires a key to decrypt the text, we assumed that the text from the Morse code could be the key for encryption, and we created the matrix for decryption. We removed all the punctuations and white spaces to make the entire text a string of characters. We wrote a c++ code to generate the plain text automatically. We got the plain text in a string as output from c++ code. We manually break it into meaningful English words and put all the punctuations back at their corresponding positions. We got the password as "ABRA_CA_DABRA,"

Q5 Decryption Algorithm

15 Points

Briefly describe the decryption algorithm used. Also mention the plaintext you deciphered. ( Use less than 350 words)

First we got the key from the Morse code which is a substitution cipher. Mapping is as follows:- -.-. = C .-. = R -.-- = Y .--. = P - = T .- = A -. = N .-.. = L ... = S .. = I For Morse code we referred to Wikipedia page(https://en.wikipedia.org/wiki/Morse_code) Decrypted text = "CRYPTANALYSIS". We have learned about Play fair cipher from this video(https://www.youtube.com/watch?v=UURjVI5cw4g). Play fair Cypher:- -> This is also kind of substitution cipher which uses diagrams. -> In this technique a 5*5 matrix is build using the key each, entry contains a English alphabet But (I and J are together). The the complete text is converted into pairs of letters. Any pair of alphabets should not have same alphabets so if both the letters are same then a dummy (different) character is introduced in between the alphabets. -> Then the relative position of alphabets is checked in the above Matrix and accordingly it is encrypted. -> To decrypt the text we just did reverse of all the operations which is done while encrypting. Matrix is as follows : C R Y P T A N L S I B D E F G H K M O Q U V W X Z Decryption Algorithm: -> First divide the whole ciphertext into pair of adjacent characters(after removing punctuation and spaces). -> For each pair, there are 3 cases to consider:- 1. If both characters are in same row, we replace each character with its preceding alphabet (considering wrapping). Replace M[i][j] by M[i][(j - 1 + 5)%5] (considering 0-based indexing) 2.If both characters are in same column, we replace the each character by its preceding character (considering wrapping). Replace M[i][j] by M[(i - 1 + 5)%5][(j] (considering 0-based indexing) 3. all other cases pos1 = (x1,y1) , pos2 = (x2,y2) Replace M[x1][y1] by M[x1][y2] Replace M[x2][y2] by M[x2][y1] (pos1 and pos2 are the positions of first and second character of the pair) -> We got a string we broke that string into meaningful words, and in many places, we needed to replace 'I' with 'J' because they have the same place in the matrix and removed dummy characters. We put all the punctuations back and we got the password as "ABRA_CA_DABRA". Mapping of Punctuations:- " = " . = . _ = _ , = , Plain Text: BE WARY OF THE NEXT CHAMBER, THERE IS VERY LITTLE JOY THERE. SPEAK OUT THE PASSWORD "ABRA_CA_DABRA" TO GO THROUGH. MAY YOU HAVE THE STRENGTH FOR THE NEXT CHAMBER. TO FIND THE EXIT YOU FIRST WILL NEED TO UTTER MAGIC WORDS THERE.
